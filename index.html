AI Agent & Full-Stack

ShopEase — AI Refund Processing Agent

Live Site Live API GitHub

An AI customer-support agent that autonomously handles e-commerce refund requests end-to-end. Built with LangGraph for agentic orchestration and Groq's Llama 3.3 70B for reasoning, the agent looks up the customer's order, validates it against store policy, and approves or denies the refund — with live streaming responses and sentiment-aware tone adjustment. Wrapped in a full storefront demo (React) so the agent can be seen operating in a real customer journey: browse, cart, checkout, support chat.

Problem Solved

Refund requests are repetitive but policy-sensitive — a bot that just says "refund approved" to everyone isn't useful, and a bot that can't actually check policy isn't trustworthy. ShopEase's agent uses real tool-calling (lookup, validate, process) so every decision is grounded in actual order data and store rules, not just plausible-sounding text.

Hard Engineering Solved

  • Fixed the LLM over-triggering refund tools on general policy questions via tightened tool-use guardrails in the system prompt
  • Found and fixed a frontend regex bug that rendered false "DENIED" decision cards from policy text mentioning the word, not an actual denial
  • Diagnosed a 401 API error to a revoked Groq key and restored service via key rotation across Render environment variables
  • Fixed SPA navigation bugs where window.location.href reloads were wiping React cart state on every route change

Active Spark Gen 7 — AI Fitness Battle App

GitHub

A real-time AI-powered fitness battle app for kids built entirely in Kotlin. Uses MediaPipe PoseLandmarker running at ~30fps on-device to detect body pose and count exercise reps automatically — no wearable needed, just a phone camera. Kids compete in live multiplayer battles where the opponent's reps sync in real time, or practice solo and earn XP.

Problem Solved

Kids often skip exercise because it's boring and solitary. Active Spark turns fitness into a competitive game — children challenge each other to squat, push-up, and jump battles. The AI camera tracks every rep automatically so there's no cheating and no manual counting. Parents get a dashboard to monitor activity.

Hard Engineering Solved

  • Fixed bitmap row-stride padding corruption in CameraX RGBA_8888 frames
  • Fixed camera restarting on every Jetpack Compose recomposition (every rep)
  • Solved leaderboard race condition using Firebase atomic Cloud Function transactions
  • Fixed matrix transform order for front-camera mirror (flip before rotate)
